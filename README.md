# ALGBaixos
O projeto ALGBaixos é uma página de carros modificados criada por mim mesmo no Instagram, com o intuito de transmitir o conhecimento e a diversidade de customização automotiva que sempre despertou meu interesse. No entanto, para aplicar meus conhecimentos em Python (Django) e em Front-end (HTML, CSS, JS, Bootstrap), resolvi criar um site para a página.
